## Requerimientos

* Node
* VSCode (opcional)
## Instalación

1. Descargar o clonar el proyecto
2. Abrir el proyecto con vscode
3. En la terminal de vscode escribir:

```bash
  npm ci
```     
    
## Ejecución

Una vez instalado, para ejecutar el servidor:

```bash
  npm run start
```
